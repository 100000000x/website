Technology stack

The tech stack that we are using consist of Javascript + React + Redux in the front-end. For backend we are using Python + Django + Django REST Framework + PostgreSQL + Celery. We deploy with docker on AWS.

What are we going to build together?

We already have some plan what to build. The list is below. However, we are limited only by our imagination and we are open to suggestion from you!

  • Create Django+React open-source boilerplate. It will cover user registration/login/profile, basic dashboard and payments (with Stripe)
  • SaaS for selling online tutorials - we will need one ;)
  • SaaS for newsletters (clone of TinyLetter)
  • SaaS for website monitoring (clone of UptimeRobot, Simpleops)
  • SaaS for scheduling social media posts
  • SaaS for website analytics (clone of Google Analytics)
  • SaaS for ... - Do you have any service in mind? just let us know!

All services will be open-source. All created services will be deployed and will be available for use - they will be paid services.
